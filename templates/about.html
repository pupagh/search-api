Many search engines track you. We don't.

It has been proven that DuckDuckGo has Microsoft trackers on their site. This is because DuckDuckGo has a partnership with Microsoft to use Bing as their source of results.

Although we use Bing results, we're being tracked, not you.

We don't use JavaScript.

Not all JavaScript is bad, but sites can use it to track you. Some websites don't look pretty at all without it, but we are an exception. All of this design can be done with CSS, a language that can only style websites.

We also want people to create search engines, too!

We have a free API that anyone can use without a key.

We're trusted by Replit!

Replit is a tool to host your site, share your code, and much more! They selected us for the Creator Fund of June 2022. We use Replit to host our site, so you can also host your own site.

We're open source!
